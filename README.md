Remap `Caps Lock` + `hjkl` (or `uhjk`, could be more convenient if you are not a vim user) to arrow keys, and `Caps Lock` + `uiop` (or `iopl`) to `Home`, `End`, `Page Up`, `Page Down`.

- [xkb](/linux/xkb)
- [xmodmap](/linux/xmodmap) - simpler but less reliable
